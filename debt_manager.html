Debt Manager

Android Application

Project details

Description
A simple Android application that helps manage personal debt. Designed to add unlimited persones, each person has their own separate debt on record. The application provides a search tool to search for specific people and debts. Also, it has the ability to collect the accounts in one place and save it as a pdf file in the downloads folder of the device. It also provides a tool to modify the data of any previously added debt, such as the description, date and amount of the debt.
